# GIK339_SamirAhmad_h24samah_Labb1

## JavaScript intro, variabler och funktioner

Detta projekt är min första labb i kursen GIK339 och handlar om grunderna i JavaScript. Syftet med labben är att jag ska öva på att skriva och köra JavaScript, förstå hur variabler och scope fungerar samt testa funktioner och jämförelser. Projektet har jag hanterat och uppdaterat via GitHub Desktop.

---

## Innehåll

Projektet består av två huvudfiler:

### index.html

* Grundläggande HTML-struktur
* Kommentarer med mina reflektioner kring hur `<script>`-taggen fungerar och hur attributet `defer` påverkar körningen
* En knapp för den extra uppgiften

### script.js

* Deklaration av variabler med `var`, `let` och `const`
* Test av scope genom block och funktioner
* Test av jämförelser med `==`, `===`, `null`, `undefined` och `NaN`
* En egen funktion `greet(name)` med parametrar och returvärde
* En extra uppgift med en enkel räknare
* Kommentarer med reflektioner och förklaringar av vad jag har lärt mig



## Kom igång

För att köra projektet själv:

1. Klona eller ladda ner projektet från GitHub.
2. Öppna projektet i **Visual Studio Code**.
3. Öppna **Live Server** (högerklicka på `index.html` och välj "Open with Live Server").
4. Öppna webbläsarens **DevTools → Console** för att se resultatet av JavaScript-koden.

Projektet hanteras via **GitHub Desktop**, vilket gör det enkelt att synka ändringar och versionera filerna.



## Vad jag har lärt mig

Under denna labb har jag lärt mig:

* Skillnaden mellan variabler deklarerade med `var`, `let` och `const`
* Hur blockscope fungerar och hur variabler är tillgängliga på olika ställen
* Hur funktioner fungerar, vad parametrar och argument är, och hur returvärden används
* Skillnaden mellan `==` och `===` samt hur specialvärden som `null`, `undefined` och `NaN` fungerar
* Hur man använder konsolen för att testa och felsöka JavaScript
* Hur man kopplar JavaScript till HTML på ett säkert sätt
* Hur jag kan använda GitHub Desktop för att hantera och versionera projekt
